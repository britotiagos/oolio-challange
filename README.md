# Oolio Challenge

## Overview

This project is a mini food ordering web application designed to list products, manage a shopping cart, and place orders. It was built with a focus on functionality and user experience.

## Features

- Product listing with images
- Cart management: add, remove, and update item quantities
- Order summary with total amount calculation
- Order confirmation
- Interactive UI elements with hover and focus states
- Discount code `HAPPYHOURS` applies 18% discount to the order total
- Discount code `BUYGETONE` gives the lowest priced item for free when buying two or more items.

## Tech Stack

- **Frontend**:
  - **TypeScript**
  - **React**
  - **Next.js**
  - **Tailwind CSS**
- **Dependencies**:
  - `@uidotdev/usehooks`
  - `clsx`
  - `tailwind-merge`
- **Development Tools**:
  - **ESLint**
  - **PostCSS**
  - **TypeScript**

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/britotiagos/oolio-challange.git
   ```
2. Install dependencies:
   ```bash
   yarn install
   ```
3. Start the development server:
   ```bash
   yarn run dev
   ```

## Scripts

- `dev`: Starts the development server with Next.js and Turbopack
- `build`: Builds the production version of the application
- `start`: Starts the production server
- `lint`: Runs ESLint to check for code quality issues
